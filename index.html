Project highlights

GWAS tutorial on DNAnexus

This tutorial aims to ease the transition between using the UK Biobank data through a local access and using the devoted platform: DNAnexus.

Since the use of DNAnexus is becoming mandatory, it is crucial to learn how to use it, even more so seeing as the comprehensive documentation can be intimidating.

The tutorial guides you through every step to perform a GWAS, as well as the very basics when using DNAnexus, which makes it beginner friendly.

App to display DNAnexus jobs

This app allows for the displaying of latest DNAnexus jobs in a convenient UI through the terminal.

Although it is possible to launch and track DNAnexus jobs purely from the terminal, doing the latter can be a bit troublesome.

With a single command, you can display, filter, and search for the most recent jobs performed in your DNAnexus project. You can also select a specific job to view the associated log.
